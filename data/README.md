There are 2 files compressed in here:

* f_medium_sessions  
  * dt (activity date)
  * profileId (site id)
  * medium (channel being used to contact the targeted audience)
  * sessions (sessions)
* f_event_sessions
  * dt (activity date)
  * profileId (site id)
  * medium (channel being used to contact the targeted audience)
  * event_category (Typically the object that was interacted with (e.g. 'Video'))
  * landing_pagepath (ignore this field)
  * country (name)
  * sessions (clicks)
  * users
